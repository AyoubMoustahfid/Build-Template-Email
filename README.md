# Build Template Email using AI Gemini

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
### Demo (Vercel): 
Open [https://build-template-email.vercel.app/](https://build-template-email.vercel.app/)

### Open in Local:
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.


## Key Envirement (.env.local):

```js
// Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT= // Enter your Convex deployment

NEXT_PUBLIC_CONVEX_URL= // Enter your public convex url

NEXT_PUBLIC_GOOGLE_CLIENT_ID= // Enter your public google client id

NEXT_PUBLIC_GEMINI_API_KEY= // Enter your public gemini api key

```
